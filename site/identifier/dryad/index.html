Identifier

DOI ➞ Landing Page ➞ Publication Resources

The use of Typed Links in the HTTP Link header is illustrated for the paper:

Ryder, T. B. and Sillett, T. S.. Data from: Climate, demography, and lek stability in an Amazonian bird, 2016. Dryad Digital Repository. https://doi.org/10.5061/dryad.5d23f.

This is a summary of the links that should be provided:

  • The Landing Page and each Publication Resource should provide a link with the cite-as relation type pointing at the HTTP DOI URI.

Link Headers for the Identifier pattern




The HTTP Link header to express identity for the Landing Page URI
http://datadryad.org/resource/doi:10.5061/dryad.5d23f is:

curl -I "http://datadryad.org/resource/doi:10.5061/dryad.5d23f"

Link: <https://doi.org/10.5061/dryad.5d23f>
      ; rel="cite-as"
