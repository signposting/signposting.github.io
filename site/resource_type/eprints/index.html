Resource Type

Repository Stable HTTP URI for Articles

The use of Typed Links in the HTTP Link header is illustrated for the paper:

De Roure, D. The future of scholarly communications. Article on University of Southampton Institutional Research Repository ePrints Soton at http://eprints.soton.ac.uk/id/eprint/370703, 2014.

This is a summary of the links that should be provided:

  • The Repository Stable HTTP URI expresses the type of the resource it identifies with a link that has the the type relation type and that points at the http://schema.org/ScholarlyArticle term from schema.org.

Link Headers for the Resource Type pattern




The HTTP Link Header to express the relation type for the Repository Stable HTTP URI
http://eprints.soton.ac.uk/id/eprint/370703 is:

curl -I "http://eprints.soton.ac.uk/id/eprint/370703"

Link: <http://schema.org/ScholarlyArticle>
       ; rel="type"
