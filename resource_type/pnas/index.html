The use of Typed Links in the HTTP Link header is illustrated for the paper:

T. Yamagishi, H. Takagishi, A. Fermin, R. Kanai, Y. Li, and Y. Matsumoto. Cortical thickness of the dorsolateral prefrontal cortex predicts strategic choices in economic games. Proceedings of the National Academy of Sciences 113, 20 (May 2016), 5582-5587. https://doi.org/10.1073/pnas.1523940113.

This is a summary of the links that should be provided:

  • The Landing Page should have a link with the type relation type with target resource info:eu-repo/semantics/humanStartPage.
  • Each Publication Resource should have a link with the type relation type, with a target resource that conveys the nature of the resource.
  • Each Bibliographic Resource should have a link with the type relation type with target resource info:eu-repo/semantics/descriptiveMetadata.

Link Headers for the Resource Typing pattern




The HTTP Link header to express the nature of the Landing Page URI
http://www.pnas.org/content/113/20/5582 is:

curl -I "http://www.pnas.org/content/113/20/5582"

Link: <info:eu-repo/semantics/humanStartPage>
      ; rel="type"

The HTTP Link header to express the nature of the Publication PDF URI
http://www.pnas.org/content/113/20/5582.full.pdf is:

curl -I "http://www.pnas.org/content/113/20/5582.full.pdf"

Link: <http://purl.org/eprint/type/ScholarlyText>
      ; rel="type"

The HTTP Link header to express the nature of the Data XSLX URI
http://www.pnas.org/content/suppl/2016/04/27/1523940113.DCSupplemental/pnas.1523940113.sd02.xlsx is:

curl -I "http://www.pnas.org/content/suppl/2016/04/27/1523940113.DCSupplemental/pnas.1523940113.sd02.xlsx"

Link: <info:eu-repo/semantics/dataset>
      ; rel="type"

The HTTP Link header to express the nature of the Supplemental Item
http://www.pnas.org/content/suppl/2016/04/27/1523940113.DCSupplemental/pnas.201523940SI.pdf is:

curl -I "http://www.pnas.org/content/suppl/2016/04/27/1523940113.DCSupplemental/pnas.201523940SI.pdf"

Link: <info:eu-repo/semantics/objectFile>
      ; rel="type"

The HTTP Link header to express the nature of the CrossRef Metadata
http://data.crossref.org/10.1073%2Fpnas.1523940113 is:

curl -I -H "Accept: application/vnd.citationstyles.csl+json" "http://data.crossref.org/10.1073%2Fpnas.1523940113"

Link: <info:eu-repo/semantics/descriptiveMetadata>
      ; rel="type"

The HTTP Link header to express the nature of the Metadata BibTeX
http://www.pnas.org/citmgr?type=bibtex&gca=pnas%3B113%2F20%2F5582 is:

curl -I "http://www.pnas.org/citmgr?type=bibtex&gca=pnas%3B113%2F20%2F5582"

Link: <info:eu-repo/semantics/descriptiveMetadata>
      ; rel="type"

http://www.pnas.org/citmgr?type=easybib&gca=pnas%3B113%2F20%2F5582 is:

curl -I "http://www.pnas.org/citmgr?type=easybib&gca=pnas%3B113%2F20%2F5582"
