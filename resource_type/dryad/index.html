Resource Type

DOI for Datasets

The use of Typed Links in the HTTP Link header is illustrated for the paper:

Ryder, T. B. and Sillett, T. S.. Data from: Climate, demography, and lek stability in an Amazonian bird, 2016. Dryad Digital Repository. https://doi.org/10.5061/dryad.5d23f.

This is a summary of the links that should be provided:

  • The Identifying DOI HTTP URI expresses the type of the resource it identifies with a link that has the the type relation type and that points at the http://schema.org/Dataset term from schema.org.

Link Headers for the Resource Type pattern




The HTTP Link header to express the resource type for the Identifying HTTP URI
https://doi.org/10.5061/dryad.5d23f is:

curl -I "https://doi.org/10.5061/dryad.5d23f"

Link: <http://schema.org/Dataset>
      ; rel="type"
