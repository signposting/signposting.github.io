Author Pattern

Signposting the Scholarly Web

To help machines discover (information about) the authors of a scholarly object, use the author link relation type to link from the object's landing page to a URI that identifies an author of the object. Author identifiers that are conveyed on links with the author relation type should be HTTP URIs chosen from namespaces that are widely supported by the community, such as ORCID, VIAF, and ISNI.

The below examples illustrate the use of author links in the HTTP Link header of HTML landing pages. But author links can also be provided in that page's HTML link element and/or in a Link Set that is discoverable from the landing page.
