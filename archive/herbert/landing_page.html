Landing Page Pattern

Signposting the Scholarly Web

Use typed links in HTTP Link headers to help machines find the resources that make up a publication, to determine what the nature of each resource is (landing page, article, dataset, ...), and what the identifying HTTP URI of the publication is.

The following relation types are used in the Landing Page pattern:

  • To convey the identifier of the publication: persistent-id
  • To connect the resources that make up a publication: item, collection
  • To convey the type of a resource that is part of a publication: type
In addition, the following attributes may be used on typed links:
  • To express the media type of the linked resource: type
  • To convey what the nature of a linked resource is (landing page, article, dataset, ...): sem-type
