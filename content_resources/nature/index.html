Publication Boundary

Publication ➞ Other Publication Resources

The use of Typed Links in the HTTP Link header is illustrated for the paper:

Oren-Suissa, M., Bayer, E. A., and Hobert, O. Sex-specific pruning of neuronal synapses in Caenorhabditis elegans. Nature 533, 7602 (Apr. 2016), 206–211. https://doi.org/10.1038/nature17977.

This is a summary of the links that should be provided:

  • The Entry Page, which is an HTML rendering of the article, should have links with the item relation type pointing at each Publication Resource.
  • Each item link should have a type attribute to express the MIME type of the resource that is the target of the link.
  • Each Publication Resource should link back with the collection relation type at the Entry Page.

Link Headers for the Publication Boundary pattern




The HTTP Link header to express boundary for the Publication HTML URI
http://www.nature.com/nature/journal/v533/n7602/full/nature17977.html is:

curl -I "http://www.nature.com/nature/journal/v533/n7602/full/nature17977.html"

Link: <http://www.nature.com/nature/journal/v533/n7602/pdf/nature17977.pdf>
       ; rel="item"
       ; type="application/pdf",
       <http://www.nature.com/nature/journal/v533/n7602/fig_tab/nature17977_SF1.html>
       ; rel="item"
       ; type="text/html",
       <http://www.nature.com/nature/journal/v533/n7602/extref/nature17977-s1.pdf>
       ; rel="item"
       ; type="application/pdf"

The HTTP Link header to express boundary for the Publication PDF URI
http://www.nature.com/nature/journal/v533/n7602/pdf/nature17977.pdf is:

curl -I "http://www.nature.com/nature/journal/v533/n7602/pdf/nature17977.pdf"

Link: <http://www.nature.com/nature/journal/v533/n7602/full/nature17977.html>
       ; rel="collection"

The HTTP Link header to express boundary for the Supplemental Item URI
http://www.nature.com/nature/journal/v533/n7602/fig_tab/nature17977_SF1.html is:

curl -I "http://www.nature.com/nature/journal/v533/n7602/fig_tab/nature17977_SF1.html"

Link: <http://www.nature.com/nature/journal/v533/n7602/full/nature17977.html>
       ; rel="collection"

The HTTP Link header to express boundary for the Supplemental Item URI
http://www.nature.com/nature/journal/v533/n7602/extref/nature17977-s1.pdf is:

curl -I "http://www.nature.com/nature/journal/v533/n7602/extref/nature17977-s1.pdf"

Link: <http://www.nature.com/nature/journal/v533/n7602/full/nature17977.html>
       ; rel="collection"
