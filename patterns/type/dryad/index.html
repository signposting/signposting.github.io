Type Scenario

Landing Page ➞ Type URI

The use of Typed Links in the HTTP Link header is illustrated for the dataset:

Ryder, T. B. and Sillett, T. S.. Data from: Climate, demography, and lek stability in an Amazonian bird, 2016. Dryad Digital Repository. https://doi.org/10.5061/dryad.5d23f.

This is a summary of the links that should be provided:

  • The Landing Page should have a link with the type relation type to convey the type of the scholarly object.
  • The Landing Page should have a link with the type relation type to convey that it is a landing page.

Link Headers for the Type pattern



The HTTP Link header to express the scholarly object type for the scholarly object with Landing Page HTTP URI
https://datadryad.org/stash/dataset/doi:10.5061/dryad.5d23f is:		 
$ curl -I "https://datadryad.org/stash/dataset/doi:10.5061/dryad.5d23f"

Link: <https://schema.org/Dataset> ; rel="type" ,
 <https://schema.org/AboutPage> ; rel="type"
